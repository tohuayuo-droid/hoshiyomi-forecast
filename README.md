# 星よみ予報

現在の天体位置を計算し、12星座ごとの一言運気を表示する静的Webアプリです。

## 解釈ロジック

`src/astrology.js` に以下を実装しています。

1. 各星座を第1ハウスとするソーラーハウス方式
2. トランジット惑星が入る相対ハウスを計算
3. 各星座の中央度数に対する主要アスペクトを判定
4. 惑星・ハウス・アスペクト・オーブ・天体速度を重み付け
5. 強い影響を複数抽出し、30文字以内の文章へ統合

文章は現在ルールベースです。AI APIは使用していません。

## 起動

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
rm -rf docs
cp -R dist docs
```

GitHub Pagesは `main` ブランチの `/docs` を公開対象にしてください。


## 解釈方針

- やさしめ・生活寄り・神秘的・ヒント調
- 不安を煽らず、注意配置も「整える機会」として表現
- 48時間先までの月の星座移動、主要アスペクト開始、順逆行切替を「次の変化」として表示
