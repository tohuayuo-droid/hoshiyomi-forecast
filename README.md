# 星よみ予報

現在の太陽・月・惑星の黄経を計算し、12星座ごとに30文字以内の「運気ポイント」を表示する静的Webアプリです。

## 特徴

- Astronomy Engineによる実際の天体位置計算
- 現在の天空図を常時表示
- 12星座別の一言運気
- 10分ごとの自動再計算
- AI API不要で再現可能なルールベース文章生成
- GitHub Pages / Cloudflare Pages / Vercelに配置可能

## 起動

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

生成された `dist` ディレクトリを公開してください。

## GitHub Pages

Viteの公開先がリポジトリ配下になる場合は、`vite.config.js` に `base: '/リポジトリ名/'` を設定してください。

## 運気ロジック

`src/main.js` 内で以下を行っています。

1. 各星座の中央度数を基準点にする
2. 現在の各惑星との主要アスペクトを判定
3. 惑星の重み・アスペクトの吉凶・正確さから影響度を算出
4. 最も強い影響のテーマから30文字以内の文章を選択

これは占星術的解釈による娯楽用ロジックです。医療・金融・法律など重要な判断には使用しないでください。
